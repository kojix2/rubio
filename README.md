## Installation

### Requirements:

**[cvlc](https://github.com/videolan/vlc)**

### Ruby Gem:

```
gem install rubio-radio
```

## Usage

```
rubio
```

Default player is `cvlc`. But you can use any command line player that can take URL of radio station as its first argument.

```
rubio --backend mpg123
```
